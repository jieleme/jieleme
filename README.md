# 自律计时器 - 网页版

这是微信小程序版本的自律计时器的网页版适配。该应用帮助用户建立自律习惯，记录清醒时间，并提供冲动干预支持。

## 功能特性

### 核心功能
- **清醒计时器**：记录用户坚持的清醒时间（天、小时、分钟、秒）
- **冲动阻断恢复进度**：使用混合曲线计算大脑恢复进度
- **成就等级系统**：根据坚持天数显示不同等级和徽章
- **在线人数统计**：模拟实时在线人数（Web版模拟数据）
- **每日打卡功能**：记录连续守护天数
- **SOS紧急干预**：提供AI鼓励和深呼吸引导

### 用户引导
- **新用户欢迎模态框**：逐步介绍应用功能
- **功能提示气泡**：介绍各个功能的用途
- **交互式引导**：帮助新用户快速上手

### 数据管理
- **本地存储**：使用浏览器的localStorage保存用户数据
- **复盘历史**：记录每次重置计时器的数据
- **连续打卡统计**：跟踪用户的连续守护天数

## 技术实现

### 架构
- **前端框架**：原生HTML5/CSS3/JavaScript
- **状态管理**：自定义状态管理系统
- **数据存储**：浏览器localStorage
- **响应式设计**：适配移动和桌面设备

### 核心组件
- **计时器模块**：实时计算和显示清醒时间
- **进度计算模块**：混合恢复曲线算法
- **等级系统**：基于坚持天数的成就系统
- **打卡系统**：日期和连续统计逻辑
- **SOS系统**：紧急干预和AI鼓励

## 使用说明

### 快速开始
1. 打开 `index.html` 文件在浏览器中
2. 开始使用计时器

### 主要操作
- **查看时间**：主界面显示清醒时间和恢复进度
- **每日打卡**：点击打卡按钮记录每日进步
- **SOS干预**：冲动来袭时点击SOS按钮获得支持
- **重置计时器**：右击重置按钮清零计时器（会保存复盘数据）

### 数据管理
- 所有数据自动保存在浏览器的localStorage中
- 清除浏览器数据会导致所有记录丢失
- 可以通过浏览器开发者工具查看和管理数据
